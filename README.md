# Github Pages にメンバー.mdを反映する手順

下記の手順で、自己紹介がmergeされている方なら誰でも作成・PRしてください。

## gem(redcarpet)をインストールする

`gem install redcarpet`

## コマンドを叩いてhtmlを作成する

`rake ghpages`

あとはローカルで `git commit` して `git push` してください。
